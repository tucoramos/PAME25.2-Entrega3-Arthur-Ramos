Este projeto é uma aplicação web de cafeteria (CleanGirl Café) feita com Next.js, que simula um cardápio moderno de comidas e bebidas, com navegação por categorias, página de detalhes de produto e sistema de favoritos. Os dados ficam em um “banco” JSON local, com rotas de API que leem e atualizam os itens, permitindo alternar o status de curtida em tempo real. A interface usa Tailwind CSS, layout com sidebar fixa, banners e grid de produtos, seguindo uma identidade visual clean e elegante.
